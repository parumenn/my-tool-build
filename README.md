## ▼ 概要
「まいつーる」は、日常的に使用する便利な機能を一つにまとめたWEBアプリケーションです。
外部のクラウドサービスに頼らず、ハードウェアからネットワーク構成まで、すべて「自前（コスト0）」で構築することにこだわっています。
あ、こんなアプリ欲しいな。とアイディアが浮かべば更新していますが完全個人で開発している為、更新頻度は期待しないでください...🙏

## ▼ 構築の背景とこだわり（開発日記）

ポートフォリオとして、本システムのインフラ構成へのこだわりを記録します。

### 1. サーバー構成：ラズパイから冗長構成へ
当初は **Raspberry Pi** 単体で運用していましたが、microSDカードへの頻繁な書き込みによるI/O体制が懸念となり、長期安定稼働のため現在は余っていたminiPCを追加。
Kubernetes (k8s)にて両ハードに対しコンテナ管理を導入し、複数台での冗長化を図っています（オマケですが）。ですがこれにより以前はアップデートの度にサイトを停止させていましたが、現在は無停止でのデプロイが可能になったのが大きいです。

### 2. 「コスト0」への挑戦
「趣味として最大限遊びつつ、維持費はかけない」をモットーにしています。
- **ハードウェア:** 自宅に眠っていたRaspberry Piとmini-PCを再利用。
- **ネットワーク:** 有料ドメインは取得せず、DDNS（MyDNS）を活用して動的IPを定期的に更新・紐付けすることで無料公開を実現しています。

### 3. セキュリティの現状と今後の課題
現在はルーターの内側（LAN内）での対策を主としています。
理想はルーターの外側にWAFやリバースプロキシを配置することですが、サードパーティの有料サービスを使わずにいかにオープンソースのみで強固な壁を自前構築するかが現在の課題です。
